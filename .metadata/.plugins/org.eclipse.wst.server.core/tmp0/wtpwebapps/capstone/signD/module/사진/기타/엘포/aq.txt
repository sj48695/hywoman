LOPPY는 'LO'TTE + 'P'OINT + 'P'AY + HA'PPY'의 합성어입니다. 고객의 삶을 행복하게 한다는 핵심가치를 해피바이러스로 이미지화하여 귀엽고 즐거운 캐릭터로 디자인하였습니다. 주요 컨셉은 ' 사람들에게 사랑과 행복을 감염시키는 해피바이러스 LOPPY!' 이며 바이러스의 기본형태인 원형으로 심플하게 디자인하였습니다. 캐릭터에 사용된 블루와 옐로우 컬러는 L.pay와 L.Point의 브랜드 아이덴티티를 유지할 수 있게 너무 벗어나지 않는 컬러로 작업했습니다. 귀여운 원형의 블루 바이러스 캐릭터에 옐로우 스카프로 포인트를 주었으며, 이를 통해 경쾌함을 나타내고자 하였습니다. 또한 캐릭터의 머리카락을 P모양으로 작업함으로써, 이 캐릭터가 L.pay와 L.Point의 캐릭터임을 나타냈습니다.
